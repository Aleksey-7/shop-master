Структура папок и файлов
index.php - главная страница интернет-магазина
/admin - папка с файлами админчасти интернет-магазина
    /assets - папка стилей и функций из Bootstrap
    /material-dashboard-master - папка шаблон интернет-магазина  из Bootstrap
    /modules - опции возможности администратора
        /products - папка с файлами по управлению товаром
            add.php - добавить товар с формой
            delete.php - удалить товар
            edit.php - редактировать данные о товаре с формой
        /categories - папка с файлами по управлению категориями
            add.php - добавить категорию с формой
            delete.php - удалить категорию
            edit.php - редактировать категорию с формой
    /parts - элементы админстраницы интернет-магазина
        nav.php - меню админстраницы
        header.php - верхняя шапка админстраницы
        footer.php - нижняя шапка админстраницы 
    /categories.php - страница категорий
    /index.php - главная админстраница
    /products.php - страница продуктов      
/parts - элементы начальной страницы интернет-магазина
    cat_nav.php - меню интернет-магазина
    header.php - верхняя шапка страницы
    footer.php - нижняя шапка страницы 
    product_card - карточка товара
/about.php - информация о нашем магазине и история
/basket.php - корзина с товарами и формочкой отправки заказа
/cat.php - краткое описание категории товара
/contacts.php - наши контакты
/product.php - краткая информация описания товара
/assets - папка с подключёнными к главной странице стилями и функциями из Bootstrap
    /css - стили из Bootstrap, подключенных к проекту
    /js - функции  из Bootstrap, подключенных к проекту
        main.js - файл с дополнительными функциями для главной страницы
/configs - все конфигурации для интернет-магазина
    db.php - подключение к базе данных
/modules - возможности элементов главной страницы
    /basket - корзина  
        add-basket.php - добавление товаров в корзину
    /products - товары
        get-more.php - добавление товаров на страницу
